# jQuery Iskaypet - Card Layout

Desafio 2 para el puesto de frontend: crear una card responsive usando html,css(sass) y un poco de js (en este caso jquery).

## Instalacion

1. Instalar dependencias:
```bash
npm install
```

2. Compilar SCSS:
```bash
npm run build-css
```

O para modo watch (desarrollo):
```bash
npm run watch-css
```
